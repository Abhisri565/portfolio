# 🚀 a.dev — Portfolio of Polisetty Sunayana Abhisri

A modern, visually stunning developer portfolio built with **React.js** and **Vite**, featuring a dark space-aurora theme with glassmorphism effects, 3D interactive cards, and smooth animations.

![Portfolio Preview](https://img.shields.io/badge/Status-Live-brightgreen) ![React](https://img.shields.io/badge/React-19.x-61DAFB?logo=react) ![Vite](https://img.shields.io/badge/Vite-8.x-646CFF?logo=vite)

---

## ✨ Features

- **3D Interactive Card** — Mouse-tracking perspective card in the hero section
- **IDE-Styled Profile Card** — Code editor themed profile display with Java syntax highlighting
- **Dark Space Theme** — Deep aurora background with glassmorphism surfaces
- **Responsive Design** — Fully responsive across all screen sizes
- **Smooth Animations** — Hover effects, glow highlights, and micro-interactions
- **Contact Form** — Functional contact form with validation
- **Resume Section** — PDF view and download functionality

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| React.js 19 | Component-driven UI |

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ installed

### Installation

```bash
# Clone the repository
git clone https://github.com/Abhisri565/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The production-ready files will be generated in the `dist/` folder.
